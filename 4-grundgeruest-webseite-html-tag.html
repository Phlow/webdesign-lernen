Grundgerüst einer HTML-Datei: Der erste HTML Befehl

HTML ist eine Markierungssprache. Einen HTML-Befehl nennt man tag oder HTML-tag. In der Regel hat jede HTML-Markierung eine Anfangs-Markierung und eine End-Markierung.

Die Markierung klammert somit etwas ein und der Browser soll es der Markierung entsprechend darstellen. Eine HTML-Markierung fängt immer mit einem < an und hört mit einem > auf. Damit der Browser die End-Markierung erkennt, bekommt diese am Anfang nach dem < noch einen / Strich. Die erste wichtige HTML-Markierung ist…

<html>

Diesen Befehl versteht der Browser so: Ey, hier fängt die Webseite an. Findet der Browser dann den End-Befehl…

</html>

…weiß der Browser: Alter, hier ist die HTML-Datei zuende.

Damit der Browser versteht, dass es sich um eine deutsche Webseite handelt, müssen wir noch mehr Informationen nachlegen. Das passiert mit einem HTML-Attribut, das einem HTML-Befehl mehr Informationen mitgibt.
